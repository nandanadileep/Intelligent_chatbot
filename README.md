| Notebook                     | Description                                                                                                                     |
| ---------------------------- | --------------------------------------------------------------------------------------------------------------------------------|
| `Intent_Classification_with_BERT.ipynb`  | Trains a BERT-based model to classify user intents (e.g., greeting, complaint, query).                              |
